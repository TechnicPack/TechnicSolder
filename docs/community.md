## Contributing

Contributions are welcome. The general workflow:

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run the test suite before submitting a pull request:

```bash
php artisan test
./vendor/bin/pint
./vendor/bin/phpstan
```

Then open a pull request against the `main` branch.
